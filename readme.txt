Crypto payment checkout by intrXn. The best way to accept cryptocurrencies.

== Description ==

Crypto payment checkout by intrXn is a payment gateway plugin for WooCommerce. Accept payment in various cryptocurrencies which is super easy to install with lowest fees.

This plugin is a feature of intrxn.com, which utilises intrxn.com API as a high performing native blockchain solution. This enables the transaction flows between crypto users and merchants seamless, cost-efficient and secure.

Read more about **intrxn.com** [here](https://intrxn.com/).

Become a intrxn.com merchant by signing up [here](https://business.intrxn.com/). 

= Key features =

* Get set up in under 5 minutes! It's never been easier!
* Accept crypto payments from your Customers with intrXn's unique Volatility Protection features!
* Get settlements in stablecoins, or in fiat in your Bank Account**
* Experience gasless transactions - so no added cost on transactions!
* Be in control with a system designed to handle end-to-end payments and Secure Accounting.
* Get insightful reports and detailed stats on payments, settlements, refunds & more.
* All of this, at a commission of ONLY 0.5% that is valid for lifetime usage (ONLY for our first 1000 Customers).

= Customer's user journey =

1. The customer checks out his or her shopping cart.
2. The customer chooses Pay with crypto via intrXn as checkout method.
3. The customer redirected to intrxn.com payment page.
4. The customer chooses his or her preferred cryptocurrency to fulfil this payment. 
5. Once the transaction is approved by intrxn.com, it will update the order status and notify the customer.

== Installation ==

= Requirements =

* [WooCommerce](https://wordpress.org/plugins/woocommerce/)
* intrxn.com [merchant account](https://business.intrxn.com/)

= Plugin installation =

1. Sign up for a [intrxn.com merchant account](https://business.intrxn.com/).
2. From your Wordpress admin panel, go to Plugins > Add New > Search plugins and search for **Crypto payment checkout by intrXn**. 
3. Select **Crypto payment checkout by intrXn** and click on **Install Now** and then on **Activate Plugin**
4. Go to your WooCommerce settings and click **Pay with crypto via intrXn** to configure the plugin.
